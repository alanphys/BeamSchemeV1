Defined protocolsÂ¶

BeamScheme has a library of standard protocol sets. Do not alter these protocols. Instead copy them and create your own sets if you wish to make changes. There is considerable variation in the implementation of the protocols by different vendors. BeamScheme attempts to provide the consensus implementation, but there is no guarantee that this will match a particular vendor.

The available predefined protocols are:

  • Default
  • All
  • AFFSAPS-JORF
  • DIN
  • IEC 60976
  • Elekta
  • Siemens
  • Varian
  • FFF-LoRes
  • FFF-HiRes
