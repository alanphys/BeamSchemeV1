2D ParametersÂ¶

BeamScheme can now calculate 2D parameters across the image. This is useful in disciplines such as Nuclear Medicine and Diagnostic Radiology as well as newer radiation beam metrics.

Available 2D beam parameters listed by their protocol invocation names are:

  • Image Statistics
    • 2D CAX Value
    • 2D Max Value
    • 2D Min Value
    • 2D Ave Value
    • 2D Min IFA
    • 2D CoM Value
    • 2D CoM Scaled
    • 2D X Res
    • 2D Y Res
    • 2D X Pixels
    • 2D Y Pixels
    • 2D X Size
    • 2D Y Size
  • 2D Flatness and Unifomity Parameters
    • 2D Uniformity NCS-70
    • 2D Uniformity ICRU 72
  • 2D Symmetry Parameters
    • 2D Symmetry NCS-70
