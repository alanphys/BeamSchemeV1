The Menu BarÂ¶

The menu bar allows you to open datasets, export profiles, edit protocols and print results. Some functions are duplicated on the toolbar in The Image Pane. Menu options can also be invoked using shortcut keys by press <alt> and the underlined character. Available menu items are:

  • The File Menu
  • The View Menu
  • The Export Menu
  • The Protocol Menu
  • The Window Menu
  • The Help Menu
