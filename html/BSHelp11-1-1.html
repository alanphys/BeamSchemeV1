Image StatisticsÂ¶

Image statistics contain basic information about the image.

Available Image statistics are:

  • 2D CAX Value
  • 2D Max Value
  • 2D Min Value
  • 2D Ave Value
  • 2D Min IFA
  • 2D CoM Value
  • 2D CoM Scaled
  • 2D X Res
  • 2D Y Res
  • 2D X Pixels
  • 2D Y Pixels
  • 2D X Size
  • 2D Y Size
