1D ParametersÂ¶

BeamScheme has a comprehensive 1D parameter list and new parameters are added on a regular basis. The file param1dfuncs.pas should be consulted for the latest parameters

Available 1D beam parameters listed by their protocol invocation names are:

  • 1D Profile Statistics
    • 1D CAX Value
    • 1D Max Value
    • 1D Max Pos
    • 1D Min Value
    • 1D Min IFA
    • 1D Average IFA
  • 1D Interpolated Parameters
    • 1D Field Edge Left 50
    • 1D Field Edge Right 50
    • 1D Field Centre 50
    • 1D Field Size 50
    • 1D Penumbra 8020 Left
    • 1D Penumbra 8020 Right
    • 1D Penumbra 9010 Left
    • 1D Penumbra 9010 Right
    • 1D Penumbra 9050 Left
    • 1D Penumbra 9050 Right
  • 1D Differential Parameters
    • 1D Left Diff
    • 1D Right Diff
    • 1D Field Centre Diff
    • 1D Field Size Diff
  • Inflection Point Parameters
    • 1D Left Infl
    • 1D Right Infl
    • 1D Field Centre Infl
    • 1D Field Size Infl
    • 1D Penumbra Infl Left
    • 1D Penumbra Infl Right
    • 1D Dose 20% FW Left
    • 1D Dose 20% FW Right
    • 1D Dose 50% FW Left
    • 1D Dose 50% FW Right
    • 1D Dose 60% FW Left
    • 1D Dose 60% FW Right
    • 1D Dose 80% FW Left
    • 1D Dose 80% FW Right
  • 1D Flatness and Uniformity
    • 1D Flatness Ave
    • 1D Flatness Diff
    • 1D Flatness Ratio
    • 1D Flatness CAX
    • 1D Uniformity ICRU
  • 1D Symmetry
    • 1D Symmetry Ratio
    • 1D Symmetry Diff
    • 1D Symmetry Ave
    • 1D Symmetry Area
  • 1D Deviation
    • 1D Deviation Ratio
    • 1D Deviation Diff
    • 1D Deviation CAX
