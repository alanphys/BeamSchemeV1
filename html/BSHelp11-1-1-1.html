2D CAX ValueÂ¶

Returns the geometric centre value of the image. If the number of points in the image are even the average of the two or four centre values is returned. If Normalise to CAX is enabled it will return 100%.

Protocol invocation name: 2D CAX Value

Normalisation affects this result.
