Default¶

The “Default” protocol is automatically loaded when BeamScheme starts. On first install this will be a copy of the All protocol. To make another protocol default perform the following steps:

  1. Select the protocol you want using the list box in the results window.

  2. Enter editing mode using the “Protocol, Edit” menu entry.

  3. Select the name “Default” in the list box in the results window.

  4. Click the save button or select the “Protocol, Save” menu entry

Make sure you do not accidentally save over another protocol.
