The Window MenuÂ¶

The Window menu allows you to maximise and/or restore the individual panes as follows:

  • Image : Maximise the Image pane

  • X Profile : Maximise the X Profile pane

  • Y Profile : Maximise the Y profile pane

  • Results : Maximise the Results pane

  • Restore : Restore all panes to their default configuration
